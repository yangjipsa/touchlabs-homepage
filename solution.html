IoT & Cloud Systems

하드웨어 제어부터 모바일 앱 연동까지, 엔드 투 엔드 IoT 생태계를 이해합니다.

  • ESP8266/WeMos D1 기반 IoT 서버 구축
  • Google Firebase 실시간 데이터 동기화
  • App Inventor 활용 모바일 모니터링 앱 제작
  • TinkerCad 시뮬레이션 기반 회로 설계

AIoT & Data Science

데이터 수집, 분석, 그리고 엣지 디바이스에서의 인공지능 구현을 학습합니다.

  • Raspberry Pi Pico 기반 AIoT 시스템
  • Python Bio-Bigdata 분석 (NumPy/Pandas)
  • 인공신경망(ANN) 이론 및 엣지 AI 구현
  • ChatGPT API 활용 지능형 서비스 연동

Advanced Maker Tech

  • Fusion 360 기반 정밀 3D 모델링
  • Eagle CAD를 활용한 전문 PCB 회로 설계

  • 3D 프린팅(Cura) 및 하드웨어 패키징

드로잉로봇

(제작키트_Arduino 드로잉로봇

라인트레이서

(제작키트_Arduino 라인트레이서

스마트조명

조도 센서로 주변 밝기를 감지해 자동으로 조명이 켜지거나, 원하는 색상으로 제어 가능한 스마트 조명을 제작하며 실생활 코딩을 경험합니다.

(제작키트_Arduino 스마트조명_
