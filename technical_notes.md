# Technical Notes for TouchLabs Website

## 자료실 파일 비밀번호 설정 방법 (2026-03-21)

구글 드라이브의 개별 파일 공유 링크에는 직접 비밀번호를 설정하는 기능이 없음.

**워크어라운드:**
1.  컴퓨터에서 공유할 파일을 **비밀번호가 설정된 ZIP 파일**로 압축한다.
2.  비밀번호가 걸린 ZIP 파일을 구글 드라이브의 `_Touchlabs_Dataroom` 폴더에 업로드한다.
3.  해당 ZIP 파일의 공유 링크를 받아 '자료실' 페이지에 등록한다.
4.  게시물 내용이나 별도의 채널을 통해 방문객에게 ZIP 파일의 비밀번호를 안내한다.
